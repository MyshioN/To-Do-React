## General info
This app is simple To-Do list app which saves your tasks in json file

## Technologies
Project is created with:
* ViteReact
* JsonServer

## Setup
To run this project do this steps:

```
-> git clone url
-> npm install
-> npm install json-server
-> npm run server
-> npm run dev
Done! Ready to use the app
```
